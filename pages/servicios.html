SERVICIOS ADICIONALES

FOTOGRAFIA

FOTOGRAFIA PROFESIONAL

El servicio mensual consta de una visita (a coordinar) con la fotógrafa, para la creación del contenido audiovisual creativo y profesional, a traves del shooting y fotoproducto.

SITIO WEB

CREACION DEL SITIO WEB

Servicio por única vez. Incluye la creación de un sitio web a medida de tus necesidades. No esta incluído el mantenimiento del sitio web ni futuras modificaciones, en caso de necesitar esos servicios, deben convenirse con el desarrollador web quien presupuestará aparte.

BRANDING

BRANDING Y RECONOCIMIENTO DE MARCA

Branding: servicio por única vez, consta de un análisis de marca, construcción de una identidad mediante varios aspectos a cargo de nuestra diseñadora gráfica, para lograr una esencia visual que se distinga fácilmente.

GESTION DE ANUNCIOS

ANUNCIOS EN RRSS Y GOOGLE

Estrategia y desarrollo de campañas publicitarias para redes sociales y Google ADS segun la necesidad de tu marca.
